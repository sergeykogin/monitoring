для запуска приложения запусти файл start_webapp.bat, открой браузер и перейди по ссылке: http://127.0.0.1:5000

Если переустановил windows: 
1. Перенеси в корень диска D папку monitoring (если она уже там - пропусти этот шаг)
2. Удали папку D:\monitoring\venv (если её там нет - пропусти этот шаг)
3. Установи python версии 3.8 ("D:\monitoring\libs_to_work\python-3.8.0-amd64.exe")
4. Запусти bat-файл install_lib.bat
5. ГОТОВО! можно пользоваться.
